### 1.conda env install OpenCV

* clone code

  > git clone https://github.com/mbeyeler/opencv-machine-learning.git

* add conda-Forge

  > conda config --add channels conda-forge

* create env

  > conda create -n Python3CV python=3.6 --file requirements.txt
